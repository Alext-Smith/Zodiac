A Python command-line programme that identifies a 
user's star sign either by name or date of birth.

## Features
- Two paths: type your sign directly or enter your birthday
- Date validation including month-specific day limits
- Displays symbol, element and description for each sign
- Option to look up multiple signs in one session

## How to run
1. Install Python 3.x
2. Open Zodiac.py in any Python editor
3. Run the file and follow the prompts

## Built with
Python 3.13 | PyCharm

## Skills demonstrated
Functions, input validation, while loops, if/elif chains,
dictionaries, string formatting
